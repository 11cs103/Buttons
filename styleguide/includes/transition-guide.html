Transition Guide 1.0 -> 2.0

We've made some major improvements to the Buttons library. In order to integrate buttons into your current project you'll need to make the following changes:

  1. Compass has been replaced with autoprefixer. Compass is not reccomended but it is still supported.
  2. Button colors are now complete independant (ex. button-primary) we no longer have classes liek button-flat-primary to achive this you now simly add button-flat button-primary
  3. Buttons styles are now independant (ex. button-flat, button-3d, etc.). You can apply these styles and they will automatically pick up the color attached to the button (ex. button-primary button-3d)
